Tyr (The Client)

A unified CLI and GUI interface for staff members. Seamlessly integrates with SSH agents and handles identity-based authentication in seconds.

Rooted in Trust

Fenrir follows a hierarchical certificate authority model, supporting everything from simple single-node setups to air-gapped Root CAs.

Hardened by Default

Support for Physical Air-Gap (Tier 3), FIDO2/WebAuthn MFA, and Proof-of-Possession (PoP) based certificate renewal.

Built for Scale

Push-based metrics reporting avoids NAT complexities. Real-time fleet health monitoring and automated anomaly detection for large-scale operations.
